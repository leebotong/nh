시작하기

클래스

1. 클래스 이름은 영문, 숫자, 대시(-)로만 구성하며 동작이 추가될 경우 언더스코어(_)를 사용합니다.
st_[name] : 스타일 타입이 있을 때 사용합니다.
cs_[value] : 변경되는 상태를 정의할 때 사용합니다.
ani_[name] : 애니메이션에 상태값을 줄 때 사용합니다.
js_[name] : 스크립트 기능이 들어간 클래스의 경우 js_를 사용하며 해당 클래스는 스타일을 적용하지 않습니다.

2. 별도 속성 값 적용시 emmit을 이용한 약어 사용으로 추가 속성을 지정합니다.
ta-center, bg-primary, c-primary, dp-flex 

								
	// st - Style Type 
	.table {}
	.table.st_borderless {}
	.table.st_striped {}
	
	.text-list {}
	.text-list.st_dot {}
	.text-list.st_dash {}
	
	// cs - Change Status
	.modal.cs_show {}
	.radio.cs_active {}
	.input.cs_disabled {}
	
	// css animation
	.modal.ani_fade-in {}
	.modal.ani_slidedown {}
	
	// js control
	.btn.js_btn-modal {}
	.btn.js_btn-modal-close {}
	.accodien.js_accordion {}
	.accodien.js_accordion {}

	// emmit
	.ta-center{}
	.bg-primary {}
	.c-primary {}
	.dp-flex {}

아이콘

svg 사용가이드

svg 사용 가이드입니다.
확장팩 다운로드 : Image To Base64, vscode-base64
1. svg 코드 전체선택 (복사X)
2. ctrl + shift + p → base64 encode 클릭 (base64로 변환됨)
3. 변환된 코드를 복사한 후 data:image/svg+xml;base64, 뒤에 불여넣기
(background:url(data:image/svg+xml;base64,복사한코드;))

아이콘

- 아이콘의 기본 크기는 24*24 입니다.
- 예외 스타일은 다중 클래스를 활용하고
상위요소의 크기를 상속 받아서 사용할 수 있습니다.
- 클래스의 이름 충돌을 방지하고 가독성을 높이기 위해 접두어(ic-)를 붙입니다.
- ic-ico-error
